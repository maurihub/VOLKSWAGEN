# 🚗 Landing Page Volkswagen Argentina - Gestión de Ventas

Esta es una landing page profesional desarrollada para la promoción y venta de la línea **Volkswagen 0km 2026**. El sitio está optimizado para captación de leads mediante integración directa con **WhatsApp**.

## 🌟 Características

* **Diseño Moderno:** Interfaz limpia siguiendo la identidad visual de Volkswagen.
* **Cuestionario Interactivo:** Sistema de pasos para filtrar las necesidades del cliente (Uso personal/trabajo y entrega de usado).
* **Conversión Directa:** Botones de consulta específicos por modelo que envían un mensaje pre-cargado al vendedor.
* **Optimización Móvil:** Diseño totalmente responsivo para una navegación fluida en celulares.
* **Botón Flotante:** Acceso rápido a WhatsApp siempre visible.

## 📱 Modelos Incluidos

El sitio cubre las principales categorías del mercado argentino:
* **Entrada y Sedanes:** Tera (sucesor del Gol), Polo Track y Virtus.
* **SUV y Crossovers:** Nivus, T-Cross y Taos.
* **Pickups:** Amarok y Saveiro.

## 🛠️ Tecnologías Utilizadas

* **HTML5** (Semántico)
* **CSS3** (Variables nativas, Flexbox y Grid)
* **JavaScript** (Lógica del cuestionario y manejo de eventos)
